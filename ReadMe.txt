Step to run automation script in postman for CoreCK

Step1: Import Collection and environment file to Postman
Step2: Select Runner and choose collecction and environment:
- Choose 'AutomationCoreCKCSV' folder in the 'Choose a collection or folder'
- Choose 'CoreCK QC' in the 'Environment'
- Select Data file : 'TestSingleRequest.csv'
Step 3: Run Automation
